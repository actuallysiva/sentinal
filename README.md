# sentinal

Sentinal is a lightweight headless utility node designed to run on old Android
devices using Termux.

## Purpose

This project exists to:
- reuse old smartphones
- run background utility services
- provide remote control and monitoring
- learn Linux-style service design

## Design principles

- PC is source of truth
- Phone is runtime only
- No secrets stored in repository
- Each service is isolated
- Simple, readable scripts

## Services (planned)

- aria2 downloader
- telegram control bot
- internet monitor
- cleanup tasks

This repository contains code and documentation only.
